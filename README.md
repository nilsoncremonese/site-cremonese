## **🚀 Como Hospedar de Graça no GitHub Pages**

O GitHub oferece hospedagem estática gratuita de forma muito simples. Siga estes passos para colocar o site no ar em minutos:

1. **Crie um repositório** no seu GitHub (ex: cremonese-engenharia).  
2. **Suba os arquivos** (index.html e image\_53aaaf.jpg) para a ramificação principal (main ou master).  
3. No GitHub, vá até as **Settings** (Configurações) do seu repositório.  
4. No menu lateral esquerdo, clique em **Pages** (dentro da seção *Code and automation*).  
5. Em **Build and deployment**, sob *Source*, mude para Deploy from a branch.  
6. Selecione a branch main (ou master) e a pasta / (root). Clique em **Save**.  
7. Aguarde cerca de 1 a 2 minutos. O GitHub gerará um link público para o seu site (ex: https://seu-usuario.github.io/cremonese-engenharia).
